# Saltmere

*A quiet village by the sea.*

Saltmere is a small living digital world designed to be visited rather than played.

There are no heroes to become.
There are no quests to complete.
There are no battles to win.

Instead, Saltmere quietly continues living whether you are there or not.

People wake each morning, work, talk, remember, celebrate, repair, rest and watch the seasons pass. The tides rise and fall, weather changes, and the village slowly develops its own history over time.

You are not the centre of this world.

You are simply another visitor who gradually becomes a familiar face.


---

## Features

- 🌅 Real-world day/night cycle
- 🌦 Dynamic weather and seasons
- 🌊 Tide simulation
- 🚶 Villagers with daily routines
- 💬 Conversations that change over time
- 🏡 Persistent world state
- 📖 Personal journal and discoveries
- 🎣 Exploration and quiet interaction
- 🌾 Handcrafted animated environments
- 💾 Automatic local save system
- 📱 Progressive Web App (PWA) support

---

## Philosophy

Saltmere is built around one simple idea:

> A digital place should feel like somewhere that already existed before you arrived.

The village does not wait for the player.

Instead, it quietly continues with everyday life.

The goal is not to complete Saltmere.

The goal is to return.

---

## Running Saltmere

Simply open `index.html` in a modern web browser.

For the best experience, serve the files using a small local web server.

Examples:

Python

```bash
python -m http.server
```

Node.js

```bash
npx serve
```

or deploy it to any static web host.

---

## Saving

Saltmere automatically stores your progress in your browser.

The village remembers:

- Visits
- Conversations
- Discoveries
- Items
- Familiarity with villagers
- World events
- Journal entries
- World state

No account or internet connection is required after installation.

---

## Installation

Saltmere can be installed as a Progressive Web App (PWA).

Once installed it behaves much like a native application and can be launched directly from your device.
